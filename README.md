# Intern Feedback Sentiment Analysis

## Objective

Analyze intern feedback to understand positive, negative, and neutral sentiments.

## Models Used

* Logistic Regression
* Transformer (DistilBERT)

## Dataset

Intern feedback containing positive, negative, and neutral reviews.

## Results

* Logistic Regression Accuracy: ~40–50%
* Transformer Accuracy: ~80%

## Insights

* Positive feedback reflects strong learning and mentorship
* Negative feedback highlights issues like poor communication and workload
* Neutral feedback shows average experiences

## Visualizations

* Confusion Matrix
* Sentiment Distribution
* Prediction Confidence

## Conclusion

This project demonstrates how NLP can help organizations analyze feedback and improve intern satisfaction.
